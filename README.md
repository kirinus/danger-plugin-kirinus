# danger-plugin-kirinus

Set of Danger rules used to check Kirinus Digital PRs.

## Usage

Install:

```sh
yarn add danger-plugin-kirinus --dev
```

At a glance:

```js
// dangerfile.js
import kirinus from 'danger-plugin-kirinus';

kirinus();
```
